CSS selectors

The CSS selectors module defines the patterns to selectelements to which a set of css rules are then applied along with their specificity.The CSS selectors module provudes us with more than 60 selectors and five combinators.

In CSS , selectors are patterns used to match,or select the elements you want to style.Selectors are also used in Javascript to enable selectind the DOM nodes to return as a NodeList.

Selectors whether used in CSS or JavaScript, enable targeting HTML elements based on their type,attributes,current states and even postion in the DOM.

CSS-Pseudo Classes

A pseudo class consist of a colon followed by the pseudo-class name.A functional pseudo-class also contains a pair of parantheses to define the arguments

Pseudo class let you apply a style to an element not only in relation to the content of document tree , but also in relation to external factors like the history of navigator

The CSS ID selector matches an element based on the value of the element's id attribute.In order for the element to be selected,it's id attribute must match exactly the value given in the selector.
